# SISTEMA-PEORRO

## requirements
* You must have installed <b>Python >= 3.10.12</b>

## install
```shell
# Python 2.7
$ sudo apt install python-tk 

# Python 3.10
$ sudo apt install python3-tk

# Python from pip
$ pip install tk

$ pip install -r requirements.txt
```

# start app
```shell
$ python index.py
```
